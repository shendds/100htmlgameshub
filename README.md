# 🎮 100 HTML Games Collection & Game Hub

Welcome to the **100 HTML Games Collection**! This project is a premium, beautifully crafted game launcher showcasing a curated bundle of 100 separate HTML5 games. All games are accessible through a single, stunning "Game Hub" interface. 

## ✨ Features

- **Massive Collection:** 100 fully playable web-based games in one place.
- **Premium UI / UX:** A "dark arcade" themed interface featuring smooth animations, glassmorphism elements, dynamic gradients, and glowing interactive game cards.
- **Instant Search & Filtering:** Real-time search functionality and category filters so players can quickly find the game they want.
- **Zero Dependencies:** Written purely with HTML5, CSS3, and Vanilla JavaScript. No node modules, build steps, or server setups required.
- **Five Categories:** Games are divided into exactly 5 engaging categories:
  - 🕹️ Arcade
  - 🧩 Puzzle
  - 🔫 Action
  - 🃏 Card
  - 🎉 Fun

## 🛠️ Technologies Used

- **HTML5:** Semantic markup, Canvas API, Web Audio API
- **Vanilla CSS:** Custom properties (variables), Grid, Flexbox, Animations
- **Vanilla JavaScript:** ES6+, DOM Management, array filtering, LocalStorage for state persistence.

## 🚀 How to Run

Because this project is built entirely on client-side web technologies, playing the games is incredibly simple!

1. **Clone or Download** the repository to your local machine.
2. Navigate to the `100 HTML GAMES` directory.
3. Open `index.html` directly in your favorite modern web browser (Chrome, Firefox, Safari, Edge).
   
   *(Note: While most games play fine directly from the file system, running a local HTTP server is recommended to avoid CORS issues with assets like images/sounds. You can use VS Code's "Live Server" extension, or run `npx serve .` / `python -m http.server` in the root folder).*
4. Select a game from the hub, click its card, and start playing!
